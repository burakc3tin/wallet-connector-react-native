## WalletConnector - React Native 

WallectConnect ile Metamask cüzdanlarına bağlanan React Native uygulaması.

⭐ Kullanıcı kayıt olarak, hesap oluşturabilir.
⭐ Kullanıcı giriş yapabilir.
⭐ Veritabanı ve auth için firebase kullanılmıştır.
⭐ AsyncStorage kullanılmıştır.
⭐ Anlık Ethereum USD verileri için Cryptocompare API kullanılmıştır.
⭐ Cüzdan bağlantısı için Wallet Connect ve Wagmi kullanılmıştır.
⭐ State yönetimi için Redux ve React Navigation kullanılmıştır.
